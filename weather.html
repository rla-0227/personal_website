Weather Report Software

Role: Individual Duration: December 2020 Category: Web Development

ABOUT

This project took me 2 weeks to complete. This project allowed me to test myself on React and working with APIs.

PROJECT ANALYSIS

Goal: The goal of this project is to create a weather report software. Users will be able to enter in the the city name to retrieve the current temperature alongside any other supplemental information about the weather.

Requirements:

  1. Add a search bar that will record the name or ZIP code and suggest possible locations or suggestions.
  2. Within the search bar, users will be able to input a location name or ZIP code and the website will display the weather for their location. The ZIP code only works for locations in the US, but text-based location search works worldwide.
  3. The website will display the weather report for the current day
  4. The weather report will be able to indicate additional information such as the sunrise/set, moonrise/set, humidity, air pressure, wind speed, visibility, and air quality. Once the user gets the results for a certain location, they will be alerted if the air quality for the location is unhealthy.
  5. The website will be able to detect the location of the user and present weather data for that location.
